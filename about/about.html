About the Developer

Hi, I am Ahmad Faraz. I built this Project Eigen as a vibe coding project to blend engineering aesthetics with practical study tracking.

The goal was not only to track preparation, but to learn clean web development separation: HTML for structure, CSS for design systems, and JavaScript for behavior, charts, storage, and cloud sync.

The current implementation focuses on AMS I and AMS II, and the same architecture is ready to expand into more math sections.

Project Style

Math + coding vibe UI, particle background, and modular front-end architecture.

Build Focus

Learning by building, with account-based sync so progress is available across devices.
